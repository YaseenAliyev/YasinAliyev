Building The Website

"My thoughts and experience while doing the project"

I started doing the project by first watching the git tutorial. After watching the video and taking notes, I started learning HTML and CSS. The video provided was not enough since it only showed us how to build it and I can't learn unless I actually understand it. So I decided to learn with a website called W3Schools. After learning the basics I decided to learn from youtube since one video taught CSS alongside HTML. After making a couple of elements, small and unfinished websites, I decided to begin building the project. After downloading couple of template websites, I realized that, in order to build a website, you must write the whole website as an HTML and only then start using CSS to style it. I built the basic headers, links, sections, paragraphs and etc. Then started to stylise using CSS. In my opinion, CSS is more complex in comparison to HTML, that's why I sometimes relied on Google or Copilot's help. After stylising and finishing the website, I made a local repository and a public one. And published the website.
